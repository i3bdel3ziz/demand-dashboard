# Demand Dashboard

A CxO-level Demand Realtime Dashboard built with React + Recharts.

---

## 🛠 Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:5173/demand-dashboard/ in your browser.

---

## 🚀 Deploy to GitHub Pages


```bash
git init
git add .
git pull git@github.com:<YOUR_USERNAME>/<repo project>
git commit -m "xxx"
git remote set-url origin git@github.com:<YOUR_USERNAME>/demand-dashboard.git
git push
npm run deploy
```

Your dashboard will be live at `https://YOUR_USERNAME.github.io/demand-dashboard/`

> Update `base` in `vite.config.js` to match your repo name before deploying.
